Service definitions describe the kinds of raw data sources and derived services for which you can manage access. Service definitions will require certain variables to be defined on resources (for example, URLs to a particular instance of a service), and a mapping of DAM roles (used to configure access on resources) to target roles (used to configure access on the target data source or service).
