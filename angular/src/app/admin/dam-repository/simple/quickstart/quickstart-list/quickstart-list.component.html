Quickstart allows users with simple controlled or registered access control policies to easily configure authorization for data in Google Cloud Storage, Big Query, or Beacon deployments via a streamlined, one-page form.

You can use Quickstart to setup as many resources as you choose. Upon finishing a Quickstart form, you may navigate to “Advanced Settings > Resources” to perform additional customisation.

Select the service where your data is stored.

Google Cloud Storage Google Cloud Storage is a RESTful online file storage web service for storing and accessing data on Google Cloud Platform infrastructure. Amazon S3 Amazon S3 or Amazon Simple Storage Service is a service offered by Amazon Web Services that provides object storage through a web service interface. BigQuery BigQuery is a fully-managed, serverless data warehouse that enables scalable analysis over petabytes of data. Beacon The Beacon API is a platform for global discovery of genomic data and variant sharing.
